How This Calculator Works

This application calculates task group distribution and queue setup based on your input configuration. It provides insights into how bins are prepared and managed across task types and ensures the calculations remain accurate for efficient queue management.

Input Fields

  • Total Bins in Grid: The total number of bins in your system.
  • Bins Level: Number of levels per bin (default is 16).
  • Total Top Cells: This value is calculated automatically as Total Bins / Bins Level.
  • Bins Available Preparations (%): Percentage of top cells available for preparation (default is 60%).

Task Type Table Calculations

  • Bins Available Preparations: Calculated as Total Top Cells * (Expected Distribution % / 100).
  • TaskGroups: Determined using Bins Available Preparations / Avg. Task per TaskGroup.
  • Max TaskGroup for Queue Setup: Calculated as (TaskGroups * Frequency Distribution) + TaskGroups.

Buttons

  • Calculate: Processes the input fields and updates the calculated values in the table.
  • Add Task Type: Adds a new row to the Task Types Distribution table for further configurations.
